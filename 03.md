# 第三章 如何提升自己行政化办公的能力



在咱们真实化的世界当中，不管是在高等院校，还是在行政单位，还是在公司架构或者在协会组织，其实我们都需要学会行政化管理的技能，才能更好的在职场的环境当中完成社会化的活动。

那么行政化管理的技能本身来说，就是要学会如何去做文案的梳理，以及如何去做活动的执行，我们在一个行政化的单位当中最需要去学习的技能，就是这两块。

那么，对于这两块的技能的学习，在很长的一段时间，其实我自己是不会的。

那么，这个不会，包括两个地方的不会。

一个不会，是不会去做行政化办公，第二个不会，是不会去做流程化的活动。

因为对于这两项东西都是没有认知的，那么在这个时候我想去提高，其实也是一件非常困难的事情。

下面给大家分享一下，我是如何去提高自己的行政化得能力的。

# 关于数据录入的案例

那么在大学的过程中，我当时是作为大学的班级班长，那么需要做班级的信息的梳理，当时不会使用 Word和Excel，当时的我仅仅只知道需要把数据录入进去，但是没有意识到我们录入的数据是非常难看的表现，就是说我没有意思的我自己做的不好。

其实还有一项技能体系，就是 PPT 的使用，那么这三项技能是咱们完成行政化办公必须要学习的核心的技能体系，那么在再往下面就是涉及到音频处理，视频处理和图片处理的技能，那么这些都是属于视觉化的呈现而这些过去的我们，其实都是不会的。

其实很多时候不是说我们不会，而是我们就没有意识到我们需要做好，这是我当时在做行政化办公的时候所遇到的问题。

# 关于活动执行的案例

那么第二次做活动的时候是我在文艺部的学干的时候。

当时去做了一个通知，当时其实就没有非常意识到应该如何去做，其实就是没有见到过这个方面的结果的范例，那么在制作和执行的时候，其实就做的非常的不好，所以我觉得我们在布置一项任务的时候，除了应该说我们的目的，更要给一些可以参考的范例，这样可以更加快速的完成既定的结果，有时候其实是为了更好的完成活动形式的突破，所以就没有范例可以参考，这个时候其实就非常考验一个人的创造能力。

后续我逐渐的提升了自己的位置，从一个学生组织的站长助理，然后逐渐的成为副站长，然后最后成为全校二十四个学院的站长的这个过程当中，我自己是去看了很多的别人所做的行政办公的东西，后续才逐渐的提升我们自己的能力。

所以很多的时候，我们不会做的主要的原因，其实是没有见过，而不是我们自己真的不会做，当你见得东西东西越多的时候，其实自己的工作的能力，会逐渐的变得更强。

真正快速的提升我们能力的关键其实是做了一份五年的材料汇编，在这个过程中，我们掌握了一套规则，通过执行这一套规则，其实就可以快速的完成我们想要完成的行政化的材料的处理。

所以在行政化的规则里面，你只要掌握这套行政化的规则，那么就会变得非常的容易。


因为我知道应该怎么去做之后，我就能够比较快速的去完成 word 和 Excel，以及 PPT 的教学，比较少做的教学是音频处理，视频处理和图片处理的技能，因为在使用的时候，下面的频率没有上门的频率高，同时完成下面的三个技能体系的训练的时间普遍也是高于基础的办公，所以需要大量的时间来训练，才可以达到比较好的结果。

大家在正式环境当中用的最多的技能还是用这个word和excel，word用的最多，需要经常去做材料汇编的地方也比较多，他需要去做数据化的呈现，所以一个个体在行政化办公当中，只要具备这两项能力就能打的比较好的开展工作。

# 对于活动预算的思考

那么当时我们在做活动的时候，其实会有一个活动的预算，那么当时其实我对于活动的预算这件事情是比较抵触啊。当然，我自己开始做真实的活动之后，我了解到一个个体如果对于自己要做的活动的预算，以及活动的流程都不清晰，那么这个活动，其实是不可能完成比较好的活动的执行的。

所以我个人觉得在做行政化办公的过程当中，其实非常需要大家去做这个两块的事情，那么第三块的还一件事情，就是要学会如何去开会。

那么在开会的过程当中，我当时去总结了一套开会的流程。

这一套开会的流程其实还不是我创造的，而是我在别人的基础上，做的优化。

在快毕业的前夕，我参加了一个老师所做的高中教学的实习，每天我们都是早上八点钟的时候去做一个晨会，然后作完晨会之后自己去做自我学习，下午的时候就去中学完成教学的一个过程。

当时我是走了这样一个流程体系的，那么在这个过程当中，其实就学会了如何去做开会的事情，那么在现在其实我们在做开会的时候，也会去这样去做。

那么大家在参加其他的会议的时候，我不知道大家有没有比较清晰的看到，就是如果说你想要去做一个会议，不管是小型的会议，还是大型的会议，还是各种其他的会议，你有没有比较清晰的思考这个会议的会前、会议的会后和会议的会中的一个问题。

所以很多时候其实不是说我们不会做事情，而是我们没有意识到需要这样去做事情。





衡量一个人是否具备行政化办公的能力，其实就是从这三个方面去做的考量！

一个方面就是他是否有作为文秘的能力，是否具备能够非常好的和轻松的去整理相应的文书材料的能力。
那么第二个事情的话就是他是否能够非常好的去筹办相应的大型、中型、小型的活动的活动执行的能力，第三个就是他是否能够非常好的去做一个就是大型小型，中型的会议的能力，具备以上能力的个体，其实就代表了一个人具备行政化管理的能力。



那么行政化的核心是在于做行政化的管理，而行政化的管理的核心主要是把自己的思想，通过文书的形式去做传递，然后再通过文书的形式去做评价。我们在做行政化管理的作用其实再后续的工作当中，不管大家是在从事高等院校的工作也好，还是行政单位的工作，还是公司化的工作，还是在协会化的工作，其实都必须去具备这些能力。


那么这些能力的具备很多情况下是要多去做事情。

其实我非常赞同的李笑来的观点，就是如果说你想要去学习某项技能体系，那么最需要去做的事情，不是去做学习，而是要在不断的使用的过程中，在错误的经验当中不断的积累经验的过程。

那么在谈到这样一个过程的时候，其实当时我在做创业训练营教学的时候，就做了这样的尝试。在错误当中不断的积累，在错误当中不断的成长，最后成为行业的领军人。

还有一个案例呀就是说渔王的儿子为什么没有把他儿子教授成为渔王？

>是因为渔王都具备了这个行业中非常多的行业的经验，每次都是直接给与行业经验，但是没有教他这个行业经验背后的逻辑是什么，所以我们的这个渔王的孩子不能成为渔王的关键，是因为他没有经过挫折所形成的经验的积累，而直接获得经验，但他不知道这个经验为什么来，那么当他在使用的时候，就不知道如何去做判断，所以说其实他没有成为渔王的关键所在，就在于他们没有这个渔王背后所形成的背后的经验。

那么在这个纬度上面的考虑问题，我们每一个孩子们不管是在学习如何去做行政化的办公，还是如何去做会议执行，还有去做大小的活动发布，没有经验的主要原因，还是因为咱们孩子们做的事情比较少。

# 针对于学生组织的思考

前一段时间我看到简书上面有人说每个学生都尽量不要加入学生会，我觉得这件事情的本身来说就是错误的，我们在这个大学当中最应该做的事情，我个人觉得最需要去做的事情就得三件事情。

一件事情是要加入学生会，学习行政化的办公的能力，第二个是要去加入图书馆学习，学习如何去学习的能力第三个其实就是要加入协会，学习专项性技能的能力。


当一个个体一旦具备了行政化办公的技能，一旦具备学习的能力，一旦具备核心性技能的能力，在这个时候你在社会上面，就能够在任何你想要从事的工作岗位当中，找到属于你所需要的工作岗位。


接下里我们继续跟大家谈一个案例。

当时在我读大学的时候，有一个学长他当时加入的是校级电视台，加入之后他就从一个学干，然后做到部长，最后坐到台上的这个过程，那么首先锻炼了他自己的行政化的能力，其次是锻炼了他拍摄、摄像和剪辑的能力，那么在这个学习过程中不断的提高自己能力的过程，他最后就以这项能力在外面开了影视传媒公司。


所以其实我想告诉大家的是，在大学的过程中，最需要去做的事情就变得很容易，我们在做学习的主要目的是为了去完成社会化的生存，包括社会化的就业和社会的创业，以及社会化的投资的这样一个过程。


# 孩子们所遇到的问题


我们现在很多孩子们其实非常缺乏的是如何去做社会化的就业的问题，还不涉及到社会化的创业，社会化的就业就是要适应这个行业，社会化的创业的本身来说是你在这个行业当中，已经做到了非常精的程度，然后具备的能够拿出作品的能力，能够完成价值体系的交换。



刚才我们说谈的这个学长，他其实就具备的创业的能力，他知道如何去帮助别人去拍照片，如何去帮助别人完成影像的拍摄，如何去帮助别人完成各种事项的执行，那么在这件事情上面做的比较好。

但是他其实有一件事情是不清楚的，就是如何去做公司化的处理，因为当时他的独公司和别人注册的公司有冲突，最后又重新去完成的公司的重新注册。

当时的他其实是没有意识到的是公司的维度，当时去做事情的时候，在后期是要去签合同和完成产品交付时候，要以公司的形式去做，关键的问题其实也是只是结构不清晰的一个过程。


现在这个学长已经在这个阶段做的比较好，那么他有这个能力去做创业的关键，其实就是一定要具备产品。


在构建公司的过程中，其实就是财务上面的问题。在财务上的结算下面其实就是三件事情，需要有产品，同时要有好的销售渠道，就要告诉别人我们在这个领域当中做得比较好，那么他在这个阶段其实没有去做平台化的营销。

现在的这个照相业务，以及发展为互联网结合起来去做的云端化的摄像和云端化的照相，他其实缺乏的是这样一点，也就是对于互联网认知不清晰，所造成的认知结构的不清醒。






当你在学生会的组织当中，从学干到部长，从部长到执行团的这个过程当中，你能够去学习如何去做事情，如何去服从，如何去领导，在这个时候，其实是在学生组织和学生协会里面，就可以快速学习行政化的能力。

当时在学校里面还会有第二学位的，其实有助于提升自己的部分的能力，在真实的环境当中所获取的能力结构其实还不是通过学习获得，而是在实践当中活动的，但你在学校里面没有实践的机会的时候。我个人还是觉得最应该去拿一个第二学历，那么在这个时候，对于自己在后期的发展过程的，不管是就业也好，创业也好，或者是做后期投资也好，都会有一定的帮助，因为你阔阔的知识的边界。


# 关于人才招聘的问题

那么在这个时候，还会有人去询问在工作之后，有没有必要去考一个硕士的学历，我自己工作之后，其实我会有种感觉，就是说我们在招聘人员的时候，在同等的能力情况下，我们普遍愿意招硕士，所以说在这个过程当中，如果说你要去就业，我觉得考一个硕士的学历或者是一个博士的学历，对于你来说会有一定的帮助，但是如果说你要创业，其实就没有了太大的关系。

像陈立飞也是在工作之后，然后去香港读的MBA，也就是相当于读一个工商管理硕士，在真实的环境当中，其实大家在读书的时候，有两种读书的方法，一种读书的方法叫做学术，也就是做学术化研究，第二种的叫做求职，叫做专业化硕士。

其实我个人觉得，大家需要去具备一个这种学历，对于后期的发展会有一定的帮助，那么不管是做学术化的研究，还是做专项化的处理，有一个这种帽子、方便自己和别人去完成职场化的社交。



也就是说你具备一定的价值，当你的个人的价值不清晰的时候，但是因为你具备学历的价值，方便去和人社交，那么你在行政化办公的维度的体系的时候会有一定的优势，所以每个个体在当下这个阶段，最好的情况下还是去拿一个硕士的学历，这个学历有保研和考研的过程，那么在保研和考研的，我个人觉得最好的方式还是属于直接去保研，而不要去考，有的时候考研会有一定的风险。


我们在做任何一件事情，叫屈利避害。

如果说你在这个过程当中，能够非常清晰的认知到你的目的是什么，比如说我们拿一个硕士学历是为了去完成社会化的一种交换那种砝码，那么在这个时候，学历比学校更重要，但果说你是为了真正的去完成自己的学术化的能力或者是专业性的技能，那么学校就比学历更重要。

所以在这个时候，其实在做任何一件事情都会有风险，不管是在考研也好，还是在像国外去完成国外的大学的升学也好，其中都是有相应的风险在里面，所以其实在真实化的世界当中，我其实很多情况下需要的是具备能力的人，但是我们在看人的时候，有的时候还是会看一下你的学历的背景，然后在后期的发展过程中，我们更多的时候，还是看一个人是否具备终身化学习的能力。



接下来我还给大家提供一个职业发展的案例。

这个人叫做滴答滴答，那么他去写了一本书，叫做跃迁，就是关于程序员的发展，主要去谈的是他过去几十年当中学习的经验，如何去做编码，如何去做管理，，那么她做了这个经验梳理之后，又去即刻即刻时间上面去做了一个专栏。然后就跳到了coinbase 里面去做技术化的这种管理，那么它的背景是某某大学的博士，当时去做直播的时候也谈到过，其实如果说做做学术，在学校里面有一个博士的学历会比较好。

但如果说在一个工作的过程当中，那么很多情况下不是说你是否具备学历，而是你是否能够有具备适应工作的能力，那么他现在加入了coinbase去做区块链的学习，那么其实他是做了很职业发展方向的调整，但是做职业发展调整的过程中，她其实有一个非常大的优势，它具备别人所不具备的就是学历化的背景，也具备别人所不具备的专向性的技能，那么它也具备自己的这个领域当中的作品。

所以，当他去跳到另外一个行业的时候，其实具备一定的社交的砝码，也就是别人能够非常好的去信任他，知道她以前有过成功的经验。




所以我想告诉大家，如果你想要去完成职业化的跳转，那么最需要去做的事情是向别人去证明，证明你在过去当中的能力，证明你现在的人，证明你能够应对未来。


因为我们每一个个体，如果你不是作为一个自由工作者，你其实需要每时每刻都需要去加入到不同的行政单位当中去，不同的公司单位，不同的协会单位，不同的高等院校的单位当中和很多人去完成团队化协作，所以其实去完成团队化的沟通完成，团队化的协作就变成了每一个人都需要去学习的一项技能体系。
