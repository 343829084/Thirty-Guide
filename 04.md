# 第四章 如何提升自己开办公司的能力



终于开始讲公司啦，我希望通过我开办公司的经历，可以帮助你快速的获得社会立足的能力。

因为在过去很长的一个阶段里面，我自己的认知被局限，一直在行政单位和高等院校里面，因为咱们是从学校这样跑过来的，然后在这个大学的过程当中经过了行政化的小型社会化的洗礼，又经过三年 世界职场化的打磨，最后才被迫开办了自己的公司，所以我在这里想要告诉任何时候想要开办公司的你，是不是真正的准备好，如果你自己没有准备好，我一定告诉你不要开设公司。

作为已经开办公司的我来说，我最想要做的事情，其实不是开办公司，而是做公司的股东，做公司的联合创始人，这样相对于来说，人会相对轻松一些，不然眼看着自己一手打造的公司，因为缺乏现金流而宣布破产，这样的局面实在不是一般人可以承受的起的。

# 公司的概念

在过去我，其实对于如何去做公司当时是一点概念都没有，在不知道如何开办公司的时候，每一次前进其实都非常的忐忑。因为前面是一片黑暗的，不像现在我对于如何开办一家公司，以及完成公司的盈利，如何如何才可以真正的做好一家公司，我现在其实都已经了然于胸。


自己开办公司的我对于刚刚毕业的孩子最想要说的一句话就是不要盲目的去开办公司，加入一家创业型的公司完成就业，其实才是你可以快速的获得成长的关键所在。


我个人觉得每一个处在学校里面的个体，或者是还正在公司上班的个体，他们对于如何去做一家公司，以及公司是什么，怎么去做公司，怎么做一家赚钱的公司。为什么要去做公司，其实是没有任何的概念的。仅仅觉得需要自己创业，不为别人打工而盲目创业，最终其实公司没有开办起来，耽误了自己在职场环境当中成长的时间。


在谈公司以前，我还是回顾一下，我自己在没有开办公司以前，对于社会的认知。


因为大家都是从高等院校里面出来的学生，都是通过考试不断的应试的方式学习出来的人，那么在这个过程中，很多人都非常方便地去从事教师性的工作，因为我们就是这样走过来的，如果说这个孩子参加了学生会，或者参加了社团的组织的人员，其实就能够非常好的在任何一家行政单位里面获得生存的空间，因为他具备相应的行政化的经验。




但是让他到一家公司里面去完成公司化的操作的时候，其实就会变得非常的困难，因为他只具备行政化办公的技能，但是不知道公司运作的法门。




当他对于公司运作的法门不清楚的时候，他就会自以为是的在付出时间完成努力，但是这个努力没有放在公司的核心部门上面，那么一旦当自己的努力不在公司的核心部门，一旦当公司解体或者当公司改组，那么在这个时候就因为自己不在核心性的部门而被无情的淘汰。



所以在一个公司的层面上面，最需要去掌握的核心的技能，就是我以前一直谈到了三个核心技能。


第一个需要懂得的就是公司整个运作的财务体系结构，第二个是要懂得如何去完成公司化的产品销售，第三个是需要非常清晰的认知到公司在买什么产品，或者在售卖什么服务，只有一个个体对于公司整个架构这三点的事情很清晰，知道公司哪个部门最赚钱，是因为什么赚钱，那么在去学习的时候就会变得非常容易。


# 公司认知上的差异

我们在做认知层面上面还有一个问题，因为大家在任何一家公司里面，经过三到五年的时间里面不断地打拼，也许具备了相应的销售的能力，也许具备的相应的产品打造技能，也许具备了相应的财务的技能，但是就是不能做一下盈利的公司。

这是为什么嘞？

因为在咱们加入到任何一家公司的时候，他的公司是具备非常清晰的盈利模式，也就是说他做的业务和服务是具备相应的的用户，而咱们重新去打造一下公司的时候，公司在社会上面的生存的根基太浅。没有人知道我们在售卖什么产品。




我们没有产品，我们也没有渠道，我们所打造的产品和我们说打造的渠道，不是别人所需要的产品，也不是别人所需要的渠道，所以在这个时候，我们就很难完成交易，一旦没有交易，公司就会面临倒闭。

# 我是如何开办公司

以上就是我跟大家谈的关于公司的结构化的问题，接下来我跟大家主要谈一谈，我自己是怎么去做公司的，以及在这个过程当中，我所看到的很多同学是如何去开公司的，最后的结局是什么样子的，方便大家更好的理解公司的本质。


在前一段时间，其实我是做过相应的分享。

那么再开公司以前，其实我自己是没有底气的，我手上只有相应的货币现金，但是对于如何开办公司，对于如何完成编程，对于产品结构也不清晰，对于销售渠道，其实都没有认知，盲目的开始上路了。




但是我当时去做公司的时候，我其实是想要去做一些事情。而那个时间节点上面，我没有记录自己的所思和所想，现在已经快忘记我当时为什么要去做这一件事情。

所以，我觉得大家每一个人在做一件事情的时候，一定要记录下来，你当时出发的最初的状态是什么样子？你为什么要做这件事情比你现在正在做的事情更重要。



那么如果说反思过去，我当时去做这件事情的主要原因是在于我个人对于未来的发展的趋势和现在所遇到实际的问题，想要给出一个可行的方案，这是我为什么想要开办一家公司的主要的原因。


当时在做在线教育的时候，我发觉虽然说我很努力，但是我缺乏流量，所有平台方他都不能给我流量，那么在这个时候，我就不能够和我的用户去完成一对一的交流的，那么在这个时候就难以实现价值最大化，所以我想通过教授编程的手段，直接去打造一个平台，然后做我们觉得有价值的产品，完成市场化的销售，这是我当时想要去做的事情。



在做公司以前，我跑了绝大多数的基金和风投机构，基本上没有任何价值，最后我觉得还是要靠自己，所以我开始打造公司，当我打造完毕公司之后，我才认知到，这个世界没有天使，只有未来利益最大化的商人，所以每一个想要成就自己一番事业的人，都不应该期望天使给你资助，一定是你在未来可以给他十倍或者百倍的回报，他才会给予你资助。


当时觉得没有人帮助我，其实是我没有认识到公司的本质，当现在已经认识到公司本质之后，我才知道过去的我，当时的自己是多么的可笑。

# 公司的名称

我们在做公司的时候，首先需要去做的事情是要去想一个公司的名字，然后登陆各个省份不同的工商局的网站去完成核名，然后你需要有一个公司的地址，这个地址可以是真实的办公地址，也可以是孵化器的虚拟地址。


# 公司的地址

哪怕是孵化器的虚拟的地址，每年也还是需要缴纳一定的虚拟地址的使用的费用，这个费用不同的地方的地址的费用得使用的费用不一样，所以其实我们在做一家公司的时候最需要拿到的也就是地址的问题。


接下来我们就需要具备公司的章程，后续下载工商局的完成股东信息的核验，基本上就完成了公司的前期任务。

# 公司的刻章

经过这个流程之后，大概一个月的时间就能够下营业执照，我在做这一件事的时候是办营业执照和刻章是一起去做的。整个时间的周期大概花了一个月的时间。

# 开办公司的对公账户

然后进行了第二步的动作是要去做各种的账户的开设。

在做各种账户开设的过程当中，主要去做税务的账户，当时在这个过程中有相应的财务人员做代办，当时在做这个财务的时候，没有非常好的和我沟通好，所以到后期的时候，我去做了财务的重新更换，那么在做财务重新更换的时候，有一个小的插曲，因为当时我觉得上一家的财务，他做的不行呀，就不能够解决，我想要解决问题。

就在我想要换一家财务的机构的时候，我不知道应该找谁，那么当时的会算账的电话就打我的电话，最后我去他的公司实地去做了考察，然后去办理了其他的业务，现在我还在使用会算账的业务。

在使用慧算账业务的过程中，负责办理我公司业务的对接姑娘现在已经换了四位，让我觉得在后续的发展过程中，也就仅仅只能认合同和认公司，我们所谓的认人的办事的流程，在这个过程中，逐渐的全部失效。

在招商银行办理对公卡的过程，拿到的是一个公司的账户凭证，后续不管是开办公积金还是社保账户，还是办理税控盘，都需要提供这个的凭证作为一种凭据，所以这个对公的账户的文本就变得非常的重要。


在获取对公账号的银行卡的时候，因为我没有招商银行个人卡，所以在这个过程中，我首先办理了个人的卡，然后才拿到了对公账户的查询卡，最终我手上就有了可以使用银行完成转账的 U 盾，从而开始开办公司的对公的业务。

# 关于公司的类型

我们在做公司的时候是两个人在做，所以说是有限责任公司。如果说你是一人做，就是有限公司。


那么在这个过程当中，我想跟大家说的是，我们在做公司的时候，最好是把个人和公司分开会比较好，因为公司的财务整个体系一旦构建起来，其实就是我们帮助国家完成管理的一个过程。


在这个时候你就要向公司注入相应的货币现金，那么一旦注入货币现金的时候，这家公司就开始运转，所以其实有时候，我们很多人都会觉得公司是自己的。而这个时候你=最需要去思考的核心问题是要认知到公司不是你的，公司是国家的。你要把公司的财务做得很清晰，一定不能够偷懒，把自己的钱和公司的钱分不清楚，不然最后害的还是你自己。

# 公积金和社保卡的办理

接下来需要做的是两件事情：一件是公积金的办理，一件事情是社保的办理。


那么当时在办理这两件事情的时候，我们其实是完全的没有经验，办理社保没有经验，在这个时候完成办理的时候，首先是需要登录相应的网站，填报了相应的表格之后，然后带上自己公司的营业执照和公章，加上办理对公业务的一卡通，就可以开始完成账户的办理。

在这个过程中，我将一卡通办理了两张，一张方便我开展业务，一张给财务完成财务的申报。

每一个个体拿到这个社保卡大概是需要两个月的时间，公积金的卡片可以使用我们招商银行的卡片直接完成更换，其实还是挺方便。

最近我发现可以通过支付宝的城市服务查看到自己的社保卡和公积金卡的信息，其实还是挺方便，告诉大家一声，可以查看自己的基本情况。

# 控税盘办理

那么接下来我们所需要去做的一些事情就是要去办理控税盘，在办理控税盘的时候，如果是孵化器的地址，需要带上相应的材料，以及房屋的发票和租聘的合同，还有一票送的基本情况和公司的基本情况，这样就可以完成控税盘的办理，大概一个星期的时间，就可以完成办理。

# 开票电脑和打印机的购买

在这个过程中，我自己选择了比较好用的电脑和打印机，解决了我的开票情况，现在就是一句话的事情，在我不知道的时候，其实花费了我很长的时间，例如选择什么电脑，选择什么打印机，如何安装打印机，如何调试打印机，如何开具发票，这些其实每一个步骤都是经验之谈，没有一线经验的个体，想要完成社会化的生存，其实是一件非常困难的事情。

现在如果说再让我去做一下公司，就会变得很轻松，但是现在我却不会再去开办公司，而是会去做公司的股东，直接投资公司，然后主要去看公司的业务，也就是从公司的财务从公司的产品和公司的销售渠道去看公司业务，我觉得现在我最大的优势就在于我自己办了公司，我能够非常清晰地看到公司的业务流程，能非常清晰地判断公司是否能够盈利。

我在完成公司化的投资的时候，比较清晰的能够去判断投入的货币现金是否可以完成增值，这是我自己没有办公司以前是不具备的能力。


现在我们在看数字货币资产的投资的时候，本身来说也是公司的产品，看公司的产品的后期也是看公司团队，也就是说在公司是否真实的打造东西，然后这些东西是否真实的靠谱，从公司的价值来看公司，从而学会合理的完成公司价值的投资。

# 别人开办公司的三个例子

第一家公司是去年的某个时间节点上面，在我旁边的有小姑娘去开办的公司，现在我不知道这家公司是否已办下来，但是现在这个小姑娘已经不见。当时因为她想要去办一个互联网公司，然后在这个过程中想要去做互联网产品，但是她最大的问题是她不懂互联网技术。

第二个想要来找相应的投资，那么当时我去看他的产品，他当时是去做一个电商的产品，我觉得做的还是可以，那么我当时就去做了询问，他想要获得2000万的货币，出让45%的股份，其实就是想要卖公司的感觉，我觉得也不一定靠谱，但是他将产品做了出来，还是值得鼓励的。

同时还有两个男孩子，想要做线上的餐饮服务的项目，我当时就问了他们，懂不懂技术，他们说不懂技术，我说你们最好还是从公众号开始做起，这样成本会小一些，

所以现在我觉得当下去做的事情的时候，还不是开公司，还是直接做公众账号，那你的公众账号做到十万粉丝，其实就具备了比较大的商业价值。

# 流量的案例

最近我看到陈立飞的正在做自己的第三本书的宣传。

因为我自己去看了他说写的三本书，当时他去香港的时候写了一本，前一段时间做新媒体的时候写了一本，最近又做了一本文档集，其实本身来说，书籍就是总结他在一段时间节点上面的人生经历。

在这个时候更想说的是人，如果一个个体始终在卖过去的经验，其实就难以成为更加优秀的自己。

# 对于行业的认知

每一个行业的知识，本身来说它仅仅是一个圆圈，当你掌握了一个圆圈之后的知识，这个时候就能够依靠这个圆圈去完成货币现金的交换，但是如果你仅仅只是去完成这个圆圈的货币交换，你的人生就会过的很无趣。

所以在这个时候，我希望大家能够跳出来去看一些东西，不要在某个领域当中一直沉迷，而对于外界的事情漠不关心，那么这不是人生应该去做的事情。

所以我们在后期所需要去完成的数据的交易的本质上面来说也就是不一样的，本身就是通过电子化的形式完成产品化的销售，所以其实大家一定要认识到的结构节点，就在于不管你是什么产品，不管你在干什么，本身来说都是一种商务，所以维持整个世界运转的核心就是商务。

线下是线下的交易，线上是线上的交易，本身都是交易，所以不要去想一些不是交易的事情，交易才是本质。

所以我们在做任何一件事情的时候，不管是使用线上的渠道也好，还是线下渠道也好，它的本质其实还是交易。





最近有很多的机构在组各种各样的活动，这些人正在干这件事情的本质其实是在于获取客流量。他们通过做大和做小的活动的过程来去获取流量的人的参与。


人流量越大他觉得自己的影响力越大，所以在这个时候，其实我们在线上的流量难以获取的时候，我们想到线下获取流量，所以其实每一个行业，每一个个体最应该去做的事情，是要想尽一切办法获取流量，然后在这个过程当中，完成产品的迭代，以及完成产品的销售。



最近我们看到锤子科技的有准备开始做产品发布会。

前段时间其实我是购买了锤子手机的。不管是锤子手机也好和还是小米手机也罢，我在使用的过程中，其实还是没有苹果手机和oppo手机好用，但是小米和锤子其实更加善于做软件，而不是做硬件。

所以实话实说我更觉得每个个体在做事情的时候，是要尽可能做自己擅长的事情，然后要学会借力，这样你在做事情的时候才能做的比较好。在商业的环境的过程当中，最终所需要去做的还是要依靠产品说话。

像小米手机和锤子手机，做手机的研发成本是非常高的，如果说所做的产品不能快速做销售，那么其实就很难地完成后期的迭代的过程，所以其实软件产品和硬件产品最大的问题还是一个产品迭代的成本问题。

# 公司的游戏

在公司的整个体系上面来谈，也就是公司的整个运作的维度来说，过去的我开过公司的时候觉得这个前面的路是黑的，现在我把公司完全做起来之后，觉得再让我开公司就变得很轻松。在后期的发展来看，其实也就不会去开公司，会以投资人的形式去做公司化的投资，而重点去关注的是公司的销售和公司的产品。这样会相对轻松一些，因为做公司的法人有很多的事情，都需要法人出面去做，时间相来说不是太自由。

所以我个人觉得，每一个现在没有开公司的个体，最应该去做的事情，是要学会用货币去投资公司，大量的去投资公司，然后在这个时候，学习别人的产品打造，学习别人的产品交易，帮助自己完成产品的交易才是最好的方式。

那么一个个体想要比较好的去玩公司，最好的方式是要自己具备流量，也就是要具备销售渠道，所以在这个时候，我个人觉得一个个体最挣钱的方式是想尽一切办法在整个互联网的时代里面获取流量，当他的流量的速度获取越快，那么其实就能够更加快速的完成产品交易，不管是卖自己的产品还是卖别人的产品，都会变成一件相对容易的事情。
